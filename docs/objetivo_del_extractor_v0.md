# Objetivo del Physical Data Extractor v0

## Propósito
Construir un script sencillo en Python que lea archivos CSV de ejemplo, valide su estructura y genere una salida limpia y uniforme.

## Entrada v0
- CSV simple
- Columnas conocidas
- Datos pequeños

## Salida v0
- CSV limpio o estructura tabular estandarizada

## Qué debe hacer
- leer archivo
- comprobar columnas obligatorias
- detectar faltantes básicos
- renombrar columnas si hace falta
- guardar salida

## Qué no debe hacer aún
- binarios
- parquet
- PyTorch
- optimización avanzada
